# Password Generator App

A desktop application built with Python and Tkinter that generates 
secure, randomized passwords and allows users to customize them 
to their needs.

## Technologies Used

- Python
- Tkinter

## Features

- Generates 12-character passwords (maximum length)
- 94-character pool including uppercase, lowercase, digits, and special characters
- 4.76×10²³ possible outcomes — probability of 2.10×10⁻²⁴ of being guessed
- Editable output — users can modify the generated password via an entry box
- Privacy toggle that masks the password with asterisks

## Setup & Installation

1. Clone the repository
2. Run the app:
   python main.py
